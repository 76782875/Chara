# dev插件

## 功能

解决在开发新模型时常常遇到的一些问题。

在开发一个模型时常常需要确定可活动部位的位置，例如旋转转轴位置、移动边界、点击范围等。使用本插件可以直接通过点击，在角色模型上展示点的位置。

## 命令

插件会接受如下命令：

| 命令                       | 功能                          |
|----------------------------|-------------------------------|
| `::dev:point:hide-all`     | 隐藏所有点                    |
| `::dev:point:${idx}:hide`  | 隐藏指定的一个点              |
